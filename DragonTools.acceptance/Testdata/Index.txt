TEST DATA
=========

This directory contains various disk images used as test data for the 
DragonTools tests.



DragonDos
---------

dragondos-empty.vdk
    Empty DragonDos filesystem.

dragondos-tunes.vdk
    N.D.U.G. Animated Tunes disk.  Downloaded from The Dragon Archive at
    http://archive.worldofdragon.org/.



OS-9
----

os9-empty.vdk
    Empty OS-9 filesystem.

os9-system.vdk
    OS-9 system disk.  Downloaded from The Dragon Archive at 
    http://archive.worldofdragon.org/.



FLEX
----

flex-system.vdk
    FLEX 1.3.2 system disk.  Downloaded from The Dragon Archive at
    http://archive.worldofdragon.org/.



RS-DOS
------

rsdos-emoty.dsk
    Empty RS-DOS filesystem.

rsdos-demos01.dsk
    Various CoCo demo programs.  Downloaded from The Dragon Archive at
    http://archive.worldofdragon.org/.
